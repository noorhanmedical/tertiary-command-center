INSTRUCTIONS — make your other app look like this mockup
=========================================================

STEP 1 — ADD THE FONT
  In your index.html <head>, add a Google Fonts link for
  "Plus Jakarta Sans" (weights 400, 500, 600, 700, 800).

STEP 2 — SET THE GLOBAL FONT + TEXT COLOR
  In your global CSS, set the body font-family stack to:
    -apple-system, SF Pro Display, SF Pro Text,
     Plus Jakarta Sans, Inter, system-ui, sans-serif
  And set body text color to a near-black: hsl(222, 47%, 11%).
  Turn on antialiasing.

STEP 3 — SET THE PAGE BACKGROUND
  On your top-level page wrapper (the div that holds the
  sidebar and main content), set a SOLID background color:
    hsl(225, 28%, 85%)   (≈ #CFD4E3, a cool blue-grey)
  No gradients. No images. Flat color edge to edge.

STEP 4 — BUILD THE TILE GRID
  Use a CSS grid that flows responsively:
    2 columns on small screens
    3 columns on medium screens
    4 columns on large screens
    20px gap between tiles
    grid container max-width 768px, centered horizontally
    page padding: 40px horizontal, 64px vertical

STEP 5 — STYLE EACH TILE
  Shape:        square (1:1 aspect ratio)
  Background:   white at 95% opacity, with a subtle backdrop blur
  Border:       1px white at 60% opacity
  Outer ring:   1px black at 5% opacity
  Corner radius: 16px
  Inner padding: 24px
  Shadow:       soft, layered — a tight 1–2px shadow plus a
                deeper drop shadow about 24px tall
  Layout:       flex column, centered, 16px gap between
                icon and label

STEP 6 — TILE ICONS (use lucide-react)
  Size:         56 × 56 px
  Stroke width: 1.5
  Color:        dark indigo (text-indigo-900, ≈ #312E81)
  Use the SAME color for every tile icon — no per-tile colors.

STEP 7 — TILE LABEL TEXT
  Font size:    14px
  Weight:       600 (semibold)
  Color:        near-black
  Letter spacing: tight
  Centered

STEP 8 — HOVER EFFECT ON TILES
  Make the tile overflow-hidden so the glow stays inside.
  On hover:
    - lift the tile up by 2px
    - fade in a violet glow layer that fills the tile
      (violet-500 at ~35% opacity, with a heavy blur,
       contained inside the rounded corners)
    - scale the icon up to 1.10×
    - tint the outer ring faint violet
  All transitions ~300ms ease-out.

STEP 9 — SIDEBAR (optional, if you keep one)
  Width:        256px
  Background:   white
  Right border: 1px light grey
  Nav items:    14px medium weight, grey text,
                rounded 8px, light hover background
  Active item:  light indigo background (indigo-50),
                dark indigo text (indigo-900)
  Section header: 11px, uppercase, wide tracking,
                  light grey, semibold

STEP 10 — TYPE SCALE (use throughout the app)
  Page title (h1):   28–34px, bold, tight tracking
  Section title (h2): 18–20px, semibold
  Body text:          14–15px, regular/medium
  Small / meta:       12px, medium, often uppercase wide-tracking
  Buttons / labels:   14px, semibold

THAT'S IT
  Those 10 steps reproduce the entire look:
  flat blue-grey background, white square glass tiles,
  dark indigo icons, and a soft violet bloom on hover.
